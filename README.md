Single-Page CV

A clean, single-page HTML resume showcasing professional experience, education, and skills in a simple and structured layout.

Features

Pure HTML (no external dependencies)

Semantic structure (header, sections, footer)

Basic SEO and Open Graph meta tags

Favicon support

How to Use

Clone the repository:

git clone https://github.com/yourusername/your-repo.git


Open the index.html file in your browser.

Edit the content to add your own information (experience, education, skills).

Why This Project

Practice structured and semantic HTML

Create a minimal, lightweight online resume

Follow the “Single-Page CV” project from roadmap.sh
